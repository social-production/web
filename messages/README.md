# Message catalogs

- `en.json` — source language (English)
- `nl.json` — Dutch

Keys use snake_case prefixes by surface: `shell_`, `settings_`, `feedback_`, `notification_`, `error_`.

Paraglide compiles these into `$lib/paraglide/messages` at build time. After editing JSON, restart the dev server.

Dutch strings were AI-assisted drafts; review before release.
